Question :

Show the Scope of variables in JS using HTML page.


   Scope in JavaScript defines accessibility of variables, objects and functions.

   There are two types of scope in JavaScript.

  1. Global Scope
  2. Local Scope
    3.


  1. Global Scope:-

    2. Variables declared outside of any function become global variables. Global variables can be accessed and modified from any function.


  2. Local Scope:-

    3. Variables declared inside any function with var keyword are called local variables. Local variables cannot be accessed or modified outside the function declaration.
